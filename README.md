# Zune Tiles Player

![Zune Tiles Player Snapshot](https://raw.githubusercontent.com/donma/zune-tiles-player/refs/heads/main/snapshot-260223.jpg)

**Zune Tiles Player** 是一個以 Zune 音樂播放器風格為靈感打造的 Web 音樂播放器 UI，模擬微軟 Zune 現正播放界面（Now Playing）並結合動態磁磚風格背景呈現。

這個專案專注於呈現復古而精緻的「動態磁磚 + 視覺化播放」介面，搭配音樂播放、動感背景與音訊資訊顯示，是一個純前端的 Zune-like 體驗範例。

---

## 🔍 專案特色

- 🎵 **Zune UI 風格**  
  重現微軟 Zune Now Playing 界面的經典視覺效果與版面設計。

- 🟦 **動態磁磚背景**  
  以隨機大小磁磚瀑布流動畫面作為背景，類似 Zune 情境式動態畫面。

- 🎶 **音樂播放資訊顯示**  
  顯示目前播放曲目、歌手、專輯名稱與進度。

- 📻 **純前端實作**  
  完全使用 HTML / CSS / JavaScript，無需後端即可運行。

---
